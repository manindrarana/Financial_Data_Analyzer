## Project Structure
- `data/` – Sample files and basic import settings  
- `notebooks/` – Scratch notebooks for quick experiments  
- `src/` – Main analysis code and helper scripts  
- `tests/` – Checks that keep the data workflows stable  
- `README.md` – Quick start guide for the project  
- `requirements.txt` – List of Python packages to install

## Configuration
- `config/` – Example environment files and connection details  
- `.env.example` – Copy to `.env` to load default keys
